# Student-Expense-Tracker
A Java-based Student Expense Tracker developed using Object-Oriented Programming concepts. This project helps students record, manage, and track their daily expenses efficiently by categorizing spending and maintaining expense records.

## Features
- Add daily expenses
- Categorize expenses
- View total spending
- Track budget and savings
- Display expense history

## Technologies Used
- Java
- OOP Concepts
- File Handling / Arrays / ArrayList (depending on your code)

## Concepts Learned
- Classes and Objects
- Constructors and Methods
- User Input Handling
- Data Management
- Conditional Statements and Loops
